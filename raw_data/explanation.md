# Raw Data
## Descripiton
The file(s) in this folder contains variables directly extracted from flight simulators to allow creation of the `.json` files in `data/`

## File Content
### X-Plane 11.50
 - AOA (degrees)
 - Weight (kg)
 - Gravity (m/s²)
 - Pitch (degrees)
 - True Airspeed (m/s)
 - Indicated Airspeed (m/s)
 - Ground Speed (kts)
 - Rho (kg/m³)
 - Lift (N)
 - Drag (N)
 - Thrust[0] N ***(NOT WORKING)***
 - Thrust[1] N ***(NOT WORKING)***
 - Cl
 - Cd
